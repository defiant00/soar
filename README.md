# Soar
One-handed chorded keyboard layout.

## Principals

* In order of preference:
    1. Home row
    2. Chorded home row
    3. Keys outside of the home row
    4. Other chords
    5. Layers
* Chords should be on the same row or column
    * Pivoting around the index or pinky finger to other rows also works
