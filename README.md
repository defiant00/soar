# Soar
One-handed chorded keyboard layout.

## Principals

* In order of preference:
    1. Home row
    2. Chorded home row
    3. Keys
    4. Chords
    5. Layers
* Chords should be on the same row or column
    * Pivoting around the index or pinky finger to other rows can also work
